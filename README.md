
# CLI Tree Tool

````bash
# build:
make
./tree --help

# install to /usr/local/bin:
make install
```

Example:
```
.
|
+---+- .git/
|   +----- COMMIT_EDITMSG
|   +----- config
|   +----- description
|   +----- HEAD
|   +----- hooks/
|   +----- index
|   +----- info/
|   +----- logs/
|   +----- objects/
|   `----- refs/
+----- .gitignore
+----- colors.h
+----- LICENSE.txt
+----- Makefile
+----- README.md
`----- tree.c
```
